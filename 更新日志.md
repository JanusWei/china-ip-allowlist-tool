# 更新日志

本文件记录中国 IP 白名单工具的重要版本变化。

## v1.0.0 — 2026-07-19

首次公开发布。

### 主要功能

- 从 APNIC 官方 delegated 数据提取 CN、HK、MO、TW 地址资源。
- 分别输出各地区 IPv4、IPv6 CIDR 文件。
- 输出四地区 IPv4、IPv6 和双栈总合集。
- 与上一版本比较新增和删除的真实地址空间。
- 校验官方 MD5、文件头记录数、报告日期和公网 CIDR。
- 提供数量门限、大比例删除保护、候选隔离和历史回滚。
- 使用纯 Python 标准库，支持 Windows、Linux、macOS。
- 中文项目结构、配置、日志和结果文件。

### 发布数据快照

- APNIC 报告日期：2026-07-18
- 源记录数：88,590
- IPv4 总合集：8,271 个 CIDR
- IPv6 总合集：3,041 个 CIDR

### 发布前验证

- 工作目录和 Git 暂存区敏感信息扫描通过。
- 日志、历史版本、候选版本和 Python 缓存已排除。
- Git 暂存内容在独立目录重新通过语法、CIDR 和 SHA-256 校验。
- 所有受校验输出固定使用 LF，避免跨平台换行转换破坏文件哈希。
